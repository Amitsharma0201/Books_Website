Computer Fundamentals

  • This book is written by Anita Goel.
  • This book is perfect for beginners in computers.
  • The book contains several topics like introduction to computers, memory, input output system, several computer hardwares, operating system and many basic concepts.
  • This book has explained each and every concept in a manner which even interests a non-computer enthusiast.

Let Us C

  • This book has been written by Yashavant Kanetkar.
  • This book has described about C programmig language in depth.
  • Book has proper explanation with simple and understandable language
  • It as a best book for beginners and advanced students alike with hundreds of sample programming and associated notes.

Operating System Concepts

  • This book has been written by Peter Baer Galvin
  • Book is famous for providing deep concepts of operating system in a simple manner.
  • This edition of the book has been thoroughly updated to include contemporary examples of how a operating system works.
  • This book contains End-of-chapter problems, exercises, review questions, and programming exercises help to further reinforce important concepts.
